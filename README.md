# AIA

## Prerequisites

- cmake
 
      $ brew instal cmake

- openCV

      $ brew install opencv

## Compile & Run

➔ Change directory to empty build folder:

     $ cd build

➔ Invoke CMake 

     $ cmake ..

➔ Compile everything 

     $ make

➔ Run programm $ cd ..

    $ ./build/app

## Test

➔ Run tests
    
    $ ./build/test_app
